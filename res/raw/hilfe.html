Help for BMI-Spy

Program to monitor your weight. It can be entered daily (given in kilograms or pounds). With your weight, birth, size and gender the body-mass-index (BMI) for you will be calculated. The weight can be shown for the last 7 days, month or quarter. Because the body mass index is proportional to the weight, the graph can also display your reduction in BMI.


Function settings

In the settings you can enter the following data:

  • Body size: Mandatory information required to calculate the BMI.
  • Gender: Mandatory information required to calculate the BMI.
  • Birthday: Mandatory information required to calculate the BMI.
  • Language: You can choose between german and english.
  • System of Units: You can choose between metric and imperial.
  • Period of time: Mandatory entry, specifies how many days to be displayed in the graph.
  • Amputations: Here you can select whether you have amputations. If you have amputations, then your BMI value corresponding to the selected amputation is calculated using a correction value.


Function: Calculate-BMI

In the screen "BMI calculator" you can enter the following data:

  • Weight: Your actual weight.
  • Age: Given in years.
  • body size: It is measured in inches/cm.
  • Gender: Indication of your gender.
With that data, calculates the BMI value and then displays an information table. By the color you can see what area is your BMI.


Function: Enter your weight

In this screen you can enter the following data:

  • Date: The current date.
  • Weight: Your actual weight.
Only one weight measure can be stored per day. An existing weight value can be overwriten by specifying the same date.
After adding a weight value, the current BMI is calculated.


Function: Weight progress to date

In this screen you will a list of all the entered date-weight pairs (if appropriate fat-, water- and muscle-percentage are displayed too). The entries are displayed sorted by date. At the top, the number of visible entries can be set. A long press opens a menu to an entry:

  • New: Opens "Enter your weight" Screen.
  • Edit: Opens "Enter your weight" Screen, This allows you to change the weight (and if appropriate fat-, water- and muscle-percentage too) for the selected date.
  • Delete: Deletes the current weight entry.

On the menu of this screen, you have the ability to import and export data. Exported data will be saved as a file "BMI_SPITZEL.csv" created on your sd-card. This file is located after the successful export in the main directory of your sd-card. When you want to import the data into the program, there musst be the file named "BMI_SPITZEL.csv" available at the (root) directory of the sd-card. Only if this file exists can data be imported.
You can transfer data to a new phone. Normally it is enough if you export the data on your old phone and then insert the sd-card into the new phone and import the data.


Function: View graph

In this screen you will see a graph of your weight trend (if appropriate fat-, water- and muscle-percentage are displayed). Currently only the course of the past week, month or quarter will be displayed. To see the next graph, tap the display or use the context-menu.
